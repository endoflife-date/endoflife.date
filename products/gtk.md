> [GTK](https://gtk.org) is a multi-platform toolkit for creating graphical user interfaces.

The GTK team releases new versions on a [regular basis](https://blog.gtk.org/2016/09/01/versioning-and-long-term-stability-promise-in-gtk/). GTK 4 and GTK 3 are [actively maintained](https://blog.gtk.org/2020/12/16/gtk-4-0/), while GTK 2 is no longer supported.

Only even-numbered minor versions are considered stable; odd-numbered minor versions are development versions.

There is no formal end-of-life policy currently published.
