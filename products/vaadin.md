> [Vaadin](https://vaadin.com) is an open-source Java framework for building web applications, combining a server-side Java UI component library (Vaadin Flow) with TypeScript/React tooling (Hilla).

Vaadin major versions get free maintenance for a limited period after release. After that, continued updates require a commercial subscription (Prime/Team or Enterprise/Ultimate), which can also extend maintenance by up to 10 additional years. See the [maintenance schedule](https://vaadin.com/maintenance) and [roadmap](https://vaadin.com/roadmap) for details.
