> [GL.iNet](https://www.gl-inet.com/) produces OpenWrt-based routers, including portable travel routers.
GL.iNet publishes firmware lifecycle information for its routers, including the latest GL.iNet firmware version, the OpenWrt version or vendor SDK base used for that firmware, and the current GL.iNet support status.
This page tracks GL.iNet's own firmware support status for travel-router models. The OpenWrt / SDK Base column describes the base used by GL.iNet firmware and does not imply that the device follows upstream OpenWrt's lifecycle. GL.iNet states that some models remain on older OpenWrt bases because of chipset SDK restrictions, and that GL.iNet patches security vulnerabilities in those older bases before releasing firmware to users.
Models marked as `Active` by GL.iNet are listed as supported. Models marked as `Ended` by GL.iNet are listed as end-of-life when GL.iNet does not publish a more specific support end date.
